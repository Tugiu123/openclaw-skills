## ₿ 加密货币策略

### 1. 价格预测押注

**策略**: 押注"不发生"

| 事件类型 | 筛选条件 | 押注方向 |
|----------|----------|----------|
| BTC > $X | No > 0.80 | 押 NO |
| ETH > $X | No > 0.75 | 押 NO |
| 特定价格 | 极端价格 | 反向操作 |

**示例**:
```
事件: "Will Bitcoin be above $150,000 by Dec 2026?"
当前: Yes = $0.95, No = $0.05
分析: 价格极度乐观，可能高估
策略: 小额押注 NO
风险: 5% | 收益: 20倍
```

### 2. 事件驱动套利

**关注事件**:
- ✅ ETF 审批结果
- ✅ 交易所被 hack
- ✅ 监管政策变化
- ✅ 巨鲸地址异动

**套利时机**:
```
官方公告前1小时 → 市场还没反应 → 埋伏
公告发布后 → 瞬间平仓 → 吃价差
```

### 3. 波动率交易

**策略**: 利用极端事件

| 事件 | 概率 | 策略 |
|------|------|------|
| 交易所破产 | No > 0.95 | 押 NO（小额） |
| 暴跌 > 50% | No > 0.90 | 押 NO（小额） |
| 暴涨 > 100% | No > 0.85 | 押 NO（中等） |

## 🏀 体育赛事策略

### 1. 散户情绪套利

**散户行为模式**:
- 赛前 2 小时集中下单
- 追热门球队
- 过度反应伤病新闻

**识别机会**:
```
热门队赔率 <- 极端（>0.85）
↓ 
反向押注冷门队
```

**示例**:
```
事件: "Will Lakers beat Warriors?"
当前: Yes = $0.88 ( Lakers)
分析: 散户过度看好 Lakers
策略: 押注 Warriors (No)
```

### 2. 赔率异常检测

**检查点**:
1. 同一比赛多个市场
2. 赔率与历史均值偏离 > 15%
3. 交易量突增但赔率不变

**操作**:
```python
if 赔率异常:
    快速反向操作
    设置止盈止损
```

### 3. 冬奥/大型赛事

**重点关注**:
- 🏆 金牌榜预测
- 🏆 特定项目冠军
- 🏆 爆冷机会

**策略**:
```
挪威/美国/加拿大 → 热门 → 押 NO
小国爆冷 → 高赔率 → 少量押 YES
```

## ⚠️ 风险控制

### 绝对规则

```
1. 单次押注 ≤ 本金 5%
2. 单日损失 ≤ 本金 15%
3. 总仓位 ≤ 80%
4. 保留 20% 现金
```

### 仓位计算

```python
# 押注 NO 示例
capital = 1000  # 本金
max_loss = capital * 0.05  # 最大单次损失 $50
no_price = 0.80  # No 价格

# 押注金额
bet_size = max_loss / (1 - no_price)
# = $50 / 0.20 = $250

# 如果赢了
payout = bet_size * no_price / (1 - no_price)
# = $250 * 0.80 / 0.20 = $1,000
```
